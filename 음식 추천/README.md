# 🍽️ 음식 추천 시스템

사용자가 원하는 속성을 선택하면 그에 맞는 음식을 추천해주는 웹 애플리케이션입니다.

## ✨ 주요 기능

- **속성 기반 추천**: 식사 시간, 상황, 맛, 국물 유무, 목적 등을 선택하여 맞춤형 음식 추천
- **매칭 점수 시스템**: 선택한 속성과 음식의 일치율을 백분율로 표시
- **랜덤 추천**: 고민할 시간이 없을 때 랜덤으로 음식 추천
- **통계 정보**: 추천 결과에 대한 통계 및 일치율 분포 제공
- **반응형 UI**: 모바일과 데스크톱에서 모두 사용 가능

## 🚀 실행 방법

### 로컬 실행

1. 필요한 패키지 설치:
```bash
pip install -r requirements.txt
```

2. 애플리케이션 실행:
```bash
streamlit run food_app.py
```

3. 브라우저에서 `http://localhost:8501` 접속

### Streamlit Cloud 배포

1. GitHub에 코드 업로드
2. [Streamlit Cloud](https://streamlit.io/cloud)에서 새 앱 생성
3. GitHub 저장소 연결
4. 메인 파일 경로를 `food_app.py`로 설정
5. 배포 완료!

## 📋 사용법

1. **속성 선택**: 왼쪽 사이드바에서 원하는 속성들을 체크박스로 선택
   - 🍽️ 식사 시간: 아침, 점심, 저녁, 간식, 야식
   - 👥 상황: 혼자, 여럿이, 회식
   - 🌶️ 맛: 매움, 안매움
   - 🍲 국물: 국물있음, 국물없음
   - 💪 목적: 다이어트

2. **추천 결과 확인**: 선택한 속성에 맞는 음식들이 일치율 순으로 표시됩니다

3. **랜덤 추천**: 속성을 선택하지 않고 "랜덤 추천 받기" 버튼을 클릭하면 랜덤으로 음식을 추천받을 수 있습니다

## 📊 결과 해석

- **일치율**: 선택한 속성 중 몇 %가 해당 음식과 일치하는지 나타냅니다
- **색상 구분**:
  - 🟢 초록색 (80% 이상): 매우 적합한 음식
  - 🟠 주황색 (60-79%): 적합한 음식
  - 🔴 빨간색 (50-59%): 보통 적합한 음식

## 📁 파일 구조

```
음식 추천/
├── food_app.py          # 메인 애플리케이션 파일
├── 음식 목록.txt        # 음식 데이터 파일
├── requirements.txt     # 필요한 패키지 목록
└── README.md           # 프로젝트 설명서
```

## 🍜 포함된 음식

현재 80여 종의 다양한 한국 음식이 포함되어 있습니다:
- 국밥류: 순대국밥, 돼지국밥, 소머리국밥, 육개장 등
- 찌개류: 김치찌개, 된장찌개, 부대찌개, 비지찌개 등
- 구이류: 삼겹살, 소불고기, 닭갈비, LA갈비 등
- 면류: 라면, 냉면, 칼국수, 우동, 파스타 등
- 기타: 피자, 치킨, 햄버거, 샐러드, 김밥 등

## 🔧 커스터마이징

### 새로운 음식 추가

`음식 목록.txt` 파일에 다음 형식으로 음식을 추가할 수 있습니다:

```
음식명: 속성1, 속성2, 속성3
```

### 속성 추가

`food_app.py`의 `meal_categories` 딕셔너리에 새로운 카테고리와 속성을 추가할 수 있습니다.
